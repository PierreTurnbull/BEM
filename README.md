# Documentation

## Convention BEM

* B -> block
* E -> Element
* M -> Modifier

```hmtl
<!-- mainlist = block -->
<ul class="main_list">
  <!-- _item = element -->
  <li class="main_list_item">
    <!-- _item_link = element -->
    <a class="main_list_item_link" href="/home">Accueil</a>
  </li>
  <li class="main_list_item">
    <!-- _item_link = element -->
    <a class="main_list_item_link" href="/about">About</a>
  </li>
  <li class="main_list_item">
    <!-- _item_link = element -->
    <a class="main_list_item_link" href="/works">Works</a>
  </li>
</ul>
```
